  • django
  • python

Django是什么

Django是一个全栈式的Python Web 框架。众所周知，Web开发由于涉及前后端，既要考虑业务逻辑又要关注用户交互与表现，更要命的是往往做后端的不熟悉前端，做前端的不理解后端与业务逻辑，如果各自采用不同的工具与框架方案合作起来更加困难重重！
而Django为你准备好了一切！提供一整套的模式与方法论，让你可以快速生成原型，也可以在后续的开发中提供强大的支持，正如其官方网站所描述的一样：

Django makes it easier to build better Web apps more quickly and with less code.

Django能做什么

Django是一个WEB框架，理论上所有的网站与网络应用都可以实现。https://www.djangosites.org 收录了大量基于Django的站点，几乎扩囊各行各业，各种不同的应用！但考虑到Django本身的技术特征，Django更适合于构建数据库驱动的WEB应用程序

Django go！

下面我们通过具体的操作来一步步跨进Django的大门！Let’go!

  • 首先你要有Python
    由于历史的原因，Python有两个版本2.X 与 3.X ,虽然现在大部分的Python应用仍然运行在2.X之上，但我还是建议您使用3.X的版本，原因是现在已经是2016啦！程序员应该面向未来！而且新的版本抛掉了历史包袱，加入了众多新的特性。你可以在Python官网下载适合你系统的Python版本。
