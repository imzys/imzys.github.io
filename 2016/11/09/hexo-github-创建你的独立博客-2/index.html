一.几个常用的操作命令

init

用于新建一个网站，如果没有设置 folder ，Hexo默认在目前的文件夹建立网站

new

新建一篇文章，layout 为可选，默认使用 _config.yml 中的 default_layout 参数代替,为文章的标题

generate

生成静态文件，也可以简单地用以下代替

deploy

部署网站，也可以简单地用以下代替

clean

清除缓存文件 (db.json) 和已生成的静态文件 (public)

二. 定制主题

Hexo可以很方便地去定制主题，如果你喜欢折腾的话，甚至可以自己制作一份独一无二的主题，当然也可以在网上下载别人做好的优秀主题,下面以NexT主题为例,讲解如何定制Hexo主题

安装主题

推荐用git的方式进行下载，这样当主题有更新时可以很方便地用git pull 进行更新。
